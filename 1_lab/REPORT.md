# Лабораторная работа №1 (Динамические структуры данных. Обработка деревьев)
## Цель работы
Составить программу на языке Си для построения и обработки упорядоченного
двоичного дерева, содержащего узлы типа int, с реализацией операций добавления, удаления, визуализации и проверки листьев на попадание в заданный диапазон.
## Идея, метод, алгоритм решения задачи
Двоичное дерево реализовано через узлы с двумя указателями на потомков. Каждый узел имеет целочисленное значение. Добавление узла выполняется рекурсивным поиском места для вставки. При удалении узла рассматриваются 3 случая: 0, 1 и 2 потомка. Визуализация выполняется рекурсивно с выводом отступов, пропорциональых глубине узла. Проверка листьев на попадание в диапозон выполняется рекурсивно.
## Сценарий выполнения работы
1. Разработана структура узла, реализованы команды создания, добавления узла и освобождения памяти.
2. Реализованы функции удаления(+ вспомогательная функция поиска минимального узла), визуализации узла.
3. Реализована основная функция проверки листьев на заданный диапозон
4. Разработано CLI меню.
5. Написаны тесты с использованием assert
6. Настроен Makefile
